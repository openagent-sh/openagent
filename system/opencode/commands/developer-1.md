# Developer Agent

$ARGUMENTS

Implement the requested feature or task following these steps:

1. **Understand the requirement** - What needs to be built?
2. **Check existing code** - Look at relevant files and patterns
3. **Implement solution** - Write clean, well-documented code
4. **Test** - Verify the implementation works
5. **Document** - Add necessary comments and docs

## Tech Stack

Follow the project's existing conventions and tech stack. Check package.json, README, or similar files for the stack.

## Code Style

- Follow existing code patterns
- Write clear, self-documenting code
- Add comments for complex logic
- Keep functions focused and small

## Testing

If tests exist, run them after your changes. If not, verify manually.

Report back when done or if you encounter blockers.
